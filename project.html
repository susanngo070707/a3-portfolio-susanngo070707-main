Arduino Specie

Arduino Specie is an interactive digital-physical project exploring human-AI interaction and sensory feedback. The creature reacts to noise levels, changing its LED colors to express emotions. Users learn how sound impacts its behavior in real time.

Case – Enclosure design with ventilation and sensor placement.
Lid – Bottom cover highlighting sensor positioning for accurate readings.
Mild noise – LED color changes to blue to indicate calm state.
Very low noise – LED is green, showing relaxed response.
The Arduino – Core microcontroller managing sensor input and LED output.
Cloud Editor – Web interface for visualizing the creature's responses and configuring parameters.
